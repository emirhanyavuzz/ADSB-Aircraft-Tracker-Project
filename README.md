**ADSB Aircraft Tracker Project**

Bu proje, havadaki uçakların konum, hız, irtifa ve kimlik bilgilerini ADS-B (Automatic Dependent Surveillance – Broadcast) sinyallerini kullanarak gerçek zamanlı olarak takip eden OpenSky API yi kullanarak daha kullanıcı dostu bir arayüzde hava araçların takibini sağlar. Uçakların yayınladığı 1090 MHz ADS-B paketlerinin içeriklerini işleyen, görselleştiren ve web tabanlı bir arayüzde sunan hafif, hızlı ve ölçeklenebilir bir takip sistemidir.
